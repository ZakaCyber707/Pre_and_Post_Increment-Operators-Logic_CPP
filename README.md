Increment Operator Logic (Pre vs Post)


📊 Logic Overview
This project demonstrates the operational sequence of increment operators in C++:
1-Pre-Increment (++a): The value is incremented first, and then the updated value is used in the expression or displayed.

2-Post-Increment (a++): The current value is used in the expression or displayed first, and then the increment occurs for the next step.
Variable State: Both operations ultimately increase the variable by 1, but the timing of that change differs.


🛠️ Features
Direct Comparison: Provides clear comments explaining the internal "first add then display" vs "first display then add" logic.
Concise Implementation: Focused entirely on a single concept to ensure clarity for learners.
